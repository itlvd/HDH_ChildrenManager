# HDH_ChildrenManager
This a tool you can control the children using the computer safer.

## Feature
### Children
- Capture Screen.
- Turn off screen after the time parent set up for relax.
- Shut down computer when expires time.
### Parent
- Setup time.
- View image capture by children's computer.
